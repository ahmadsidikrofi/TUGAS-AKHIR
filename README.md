Easy step to run locally

1. Install Composer

   ```bash
   Composer install
   ```

2. Copy the `.env.example` to `.env` and update the variables.

   ```bash
   cp .env.example .env
   ```

3. Generate key for `.env`

   ```bash
   php artisan key:generate
   ```
   
4. Push the database locally

   ```bash
   php artisan migrate
   ```
   
5. Start the server

   ```bash
   php artisan serve
   ```
